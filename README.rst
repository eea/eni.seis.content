==============================================================================
eni.seis.content
==============================================================================

ENI SEIS Content-types


Installation
------------

Install eni.seis.content by adding it to your buildout::

    [buildout]

    ...

    eggs =
        eni.seis.content


and then running ``bin/buildout``
